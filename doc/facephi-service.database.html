facephi-service.database

Database API

db-spec

(db-spec config)
Generates a database specifification map that you can pass to
functions that query the database.

get-api-key

(get-api-key db hashed_api_key)
Gets a single API key

get-user

(get-user db username)
Finds an existing user

get-user-by-identification

(get-user-by-identification db identification)
Finds an existing user by his identification

get-user-by-identification-tx

(get-user-by-identification-tx db-spec identification)
Wraps a database query so we can filter the template bytes[] accordingly.

get-user-by-username-tx

(get-user-by-username-tx db-spec username)
Wraps a database query so we can filter the template bytes[] accordingly.

save-api-key!

save-user!

(save-user! db username is_active face identification)

save-user-log!

(save-user-log! db username identification transaction_type transaction_result)
Creates a new log record

unlock-user!

(unlock-user! db username)
Unlocks an existing user
